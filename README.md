# British-Airways-Customer-Booking-Prediction
Machine Learning project predicting customer booking behavior using Python and Random Forest.


## **📌 Project Overview**

This project was completed as part of the **British Airways Data Science Job Simulation (Forage)**. It leverages **Machine Learning** to predict whether a customer will complete a flight booking based on travel preferences, booking behavior, and historical airline data.

By applying the **Random Forest Classifier**, the model identifies the most influential factors affecting booking decisions and provides actionable business insights for improving customer conversion.

---

## **🎯 Project Objectives**

- **Predict customer booking completion (Yes/No)**
- **Analyze customer booking behavior**
- **Build a robust and interpretable Machine Learning model**
- **Evaluate model performance using Cross-Validation**
- **Identify the most important booking predictors**

---

## **📊 Dataset Features**

The dataset includes the following customer and travel attributes:

- **👥 Number of Passengers**
- **💳 Sales Channel**
- **✈️ Trip Type**
- **⏳ Purchase Lead Time**
- **🏨 Length of Stay**
- **🕒 Flight Hour & Day**
- **🌍 Route & Origin**
- **🎒 Extra Baggage Preference**
- **💺 Preferred Seat Selection**
- **🍽️ In-Flight Meal Preference**
- **⌛ Flight Duration**

### **🎯 Target Variable**

- **booking_complete**
  - **0 = Booking Not Completed**
  - **1 = Booking Completed**

## **🤖 Machine Learning Model**

### **Random Forest Classifier**

**Why Random Forest?**

- **Handles both numerical and categorical features effectively**
- **Reduces overfitting through ensemble learning**
- **Provides Feature Importance for interpretability**
- **Delivers strong predictive performance on structured datasets**

---

## **📈 Model Performance**

- **Accuracy:** 0.8559
- **Cross-Validation:** Performed to ensure model stability and reliability
- **Feature Importance:** Used to identify the strongest predictors of customer booking behavior

---

## **🔑 Key Insights**

- **Purchase Lead Time** is one of the strongest indicators of booking completion.
- **Flight Duration** significantly influences customer decisions.
- **Ancillary Services** such as baggage, seat selection, and meals contribute to booking conversion.
- **Customer engagement and travel preferences** play an important role in predicting booking behavior.
